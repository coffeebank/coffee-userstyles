## Get Started

**Step 1:** Download [Stylus](https://github.com/openstyles/stylus#stylus) *(Chrome, Firefox, Safari, Edge, Opera, ...)*

**Step 2:** Find the userstyle you want below and click **`"Download"`**

**Step 3:** Click **`"Install Style"`**
